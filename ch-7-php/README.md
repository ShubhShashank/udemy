## Lecture 171: Variables

* Variable Syntax
 
 * no #'s at beginning of variable

 * `$var="\"This var is set to a string, inside double quotations\"";`
 
* to display

 * `echo $varName`
 
* Concatenate: use "." instead of "+"

 *`echo $var1.$var2;`

 * `$name="devon";
    $var="name";
    echo $$var;`
